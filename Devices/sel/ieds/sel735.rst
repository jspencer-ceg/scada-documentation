*****************************
DNP Default Variation Options
*****************************

The default variation of Analog Inputs can be changed using the ``DNP DVAR`` commands below. These changes affect all DNP sessions on the meter.

- :samp:`DNP DVAR {VAR_NUM}` allows the default variation for Object 30 queries set to 1 through 5, as defined in the table below. Replace :samp:`{VAR_NUM}` with the variation number. This requires Access Level 2 access.
- :samp:`DNP DVAR C` (Clear) resets the Object 30 default variation to 4, and Object 32 variation to 2. This requires Access Level 2 access.
- :samp:`DNP DVAR S` (Show) returns the current variation. This requires Access Level 1 access.
- :samp:`DNP DVAR H` (Help) returns the command syntax. This requires Access Level 1 access.

The following table lists the Object 32 default variation that is linked to each Object 30 default variation.

+-------------------------------------------------+--------------------------------------------------------+
| Object 30 Variation                             | Object 32 Variation                                    |
+=================================================+========================================================+
| 1 (32-bit with flag)                            | 3 (32-bit with time)                                   |
+-------------------------------------------------+--------------------------------------------------------+
| 2 (16-bit with flag)                            | 4 (16-bit with time)                                   |
+-------------------------------------------------+--------------------------------------------------------+
| 3 (32-bit without flag                          | 1 (32-bit without time)                                |
+-------------------------------------------------+--------------------------------------------------------+
| 4 (16-bit without flag)                         | 2 (16-bit without time)                                |
+-------------------------------------------------+--------------------------------------------------------+
| 5 (short floating point analog input with flag) | 7 (short floating point analog change event with time) |
+-------------------------------------------------+--------------------------------------------------------+

The standard for CEG projects is to use Object 30 variation 1. Use the :samp:`DNP DVAR 1` command to set this in the meter.

Procedure
=========

1. Open AcSELerator QuickSet.
#. Select :menuselection:`Communications --> Parameters` or :kbd:`Ctrl+R` to open :guilabel:`Communication Parameters`.
#. For :guilabel:`Active Connection Type`, choose :guilabel:`Network`.
#. Set the :guilabel:`Host IP Address` of the meter.
#. Select :guilabel:`OK`.
#. Select :menuselection:`Communications --> Terminal` or :kbd:`Ctrl+T` to open a terminal window.
#. Select :kbd:`Enter`. The terminal should show an ``=`` prompt.
#. Enter :samp:`ACC` to enter Access Level 1. The default password is ``OTTER``.
#. Enter :samp:`2AC` to enter Access Level 2. The default password is ``TAIL``.
#. Enter :samp:`DNP DVAR 1` to change the object 30 default variation to 1. Enter :samp:`y` to confirm the change.
#. Enter :samp:`QUI` to return to Access Level 0.

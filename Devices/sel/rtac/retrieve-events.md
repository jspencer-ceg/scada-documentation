####################
Retrieve event files
####################

#. Open a web browser and go to the web page for the project's RTU RTAC. For the address, use:

      :samp:`https://{ADDRESS}`

   Replace :samp:`{ADDRESS}` with the IP address or the FQDN of the RTAC.

#. Log into the RTAC.

#. On left sidebar, select :guilabel:`Event Collection` under the :guilabel:`Reports` header.

#. If desired, use the :guilabel:`Event Timestamp Filter` or :guilabel:`Device Filter` to narrow the results. Select :guilabel:`Apply Filter` to apply the filters.

#. In the table of events, select the individual events you wish to retrieve.

#. Select :guilabel:`Download` to download the events.

#. Events will be collected into a ``.zip`` file and downloaded to your downloads folder.

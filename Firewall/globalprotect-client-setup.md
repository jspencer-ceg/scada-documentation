################################
Create an authentication profile
################################

Step 1. Add an authentication profile.
======================================

#. Navigate to :menuselection:`Device --> Authentication Profile`.
#. Select :guilabel:`Add`.

Step 2. Configure the authentication profile.
=============================================

#. Give the profile a :guilabel:`Name`:

      :samp:`Local`

#. On the :guilabel:`Authentication` tab, set :guilabel:`Type` to
   :guilabel:`Local Database`.

#. Go to the :guilabel:`Advanced` tab and on the :guilabel:`Allow List`,
   select :guilabel:`Add`.

#. Select :guilabel:`all`.

#. Select :guilabel:`OK` to save the authentication profile.
